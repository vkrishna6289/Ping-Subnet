Ping Subnet is a GUI-based network discovery and host monitoring application built using Python, PyQt, and Scapy. The tool performs subnet-wide scanning by sending ICMP and ARP requests to detect active devices on a local network, measure round-trip time (RTT), and display live host information in an interactive desktop interface. Designed as a lightweight network diagnostics utility, it allows users to input an IP range or subnet, perform real-time scans, view responsive devices, and analyze connectivity across the network. The application uses multithreading for efficient parallel scanning and integrates low-level packet crafting for accurate network probing. This project serves as a practical implementation of Layer 2 and Layer 3 networking concepts, including ARP resolution, ICMP echo handling, and packet-based device discovery, making it suitable for network analysis, troubleshooting, and infrastructure monitoring.
